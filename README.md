# Projeto
O Projeto "*Portal da Transparência do Transporte*" é de cunho educacional e visa a aplicação de conceitos estudados durante a disciplina de Inteligência Artificial.
Repositório backend: *https://github.com/ViniciusAndreGoncalves/ia-transporte-api.git*
## Tecnologias
- Front-End: React.js, TailwindCSS.
- Bibliotecas: ReactFLow (Geração e manipulação de grafos), Shadcn (componentes prontos)

## Instalação
````
clone https://github.com/MiguelMartini/Portal_da_Transparencia.git
cd PORTAL_DA_TRANSPARENCIA
(edite o env para o backend)
npm install
npm run dev
````
